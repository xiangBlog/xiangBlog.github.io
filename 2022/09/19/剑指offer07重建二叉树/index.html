剑指offer07重建二叉树

题目描述：

本题目较为简单的思路是使用递归去求解。

​ 题目中给了前序遍历结果以及中序遍历的结果，按照这两个遍历的性质，前序遍历的顺序是根结点->左子树->右子树，而中序遍历的顺序是左子树->根节点->右子树。因此我们每一次从前序遍历得到根节点，然后从中序遍历中得到根节点的位置，得到该位置之后，左边的就是左子树，右边的就是右子树，然后接着对两边进行递归求解。

​ 上述获取根节点的位置，可以使用一个map来达到O(1)的复杂度，因为题目中提及无重复数字。

递归需要确定的两个步骤：

  1. 递归参数

    ​ 这部分我们每一次递归需要得到左子树以及右子树的范围，而且每一次递归我们需要得到根节点的值，我们可以根据前序遍历的第一个数字作为跟节点，然后通过map来获取根节点在中序的位置，这样就得到了左子树以及右子树的范围，这样只是得到的中序数组中的两个子树的范围，还需要确定前序数组中两个子树的范围，可以根据左子树节点的数目来计算得出前序数组的两个子树的范围。

    ​ 这样每一次我们需要传入的递归参数就是，子树在前序遍历的范围（两个下标），子树在中序遍历的范围（两个下标）

  2. 递归的结束条件

    当左索引大于右索引，直接范围null

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    Map<Integer,Integer> map = new HashMap<>();
    int[] preorder;
    int[] inorder;
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        this.preorder = preorder;
        this.inorder = inorder;
        int len = preorder.length;
        if(len==0){
            return null;
        }
        if(len==1){
            return new TreeNode(preorder[0]);
        }
        for(int i = 0;i<len;i++){
            map.put(inorder[i],i);
        }
        return myBuildTree(0,len-1,0,len-1);
    }

    public TreeNode myBuildTree(int preLeft,int preRight,int inoLeft, int inoRight){
        
        if(inoLeft>inoRight){
            return null;
        }else if(inoLeft == inoRight){
            return new TreeNode(inorder[inoLeft]);
        }
        TreeNode root = new TreeNode(preorder[preLeft]);
        int index = map.get(preorder[preLeft]); //获取跟节点的下标
        int leftNumber = index - inoLeft;
        int rightNumber = inoRight - index;
        root.left = myBuildTree(preLeft+1,preLeft+leftNumber,inoLeft,index-1);
        root.right = myBuildTree(preLeft+leftNumber+1,preRight,index+1,inoRight);
        return root;
    }
}
