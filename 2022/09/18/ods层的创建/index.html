1.用户行为数据

1.1 创建日志表ods_log

创建表格

drop table if exists ods_log;
create external table ods_log(
    line string
)
partitioned by (dt string)
location '/warehouse/gmall/ods/ods_log';

加载数据

load data inpath '/origin_data/gmall/log/topic_log/2021-06-14' into table ods_log partition(dt='2021-06-14');

1.2 Shell中单引号和双引号区别

在/home/atguigu/bin创建一个test.sh文件

[atguigu@hadoop102 bin]$ vim test.sh

在文件中添加如下内容

#!/bin/bash
do_date=$1

echo '$do_date'
echo "$do_date"
echo "'$do_date'"
echo '"$do_date"'
echo `date`

查看执行结果

[atguigu@hadoop102 bin]$ test.sh 2021-06-14
$do_date
2021-06-14
'2021-06-14'
"$do_date"
2021年 06月 18日 星期四 21:02:08 CST

总结

（1）单引号不取变量值

（2）双引号取变量值

（3）反引号`，执行引号中命令

（4）双引号内部嵌套单引号，取出变量值

（5）单引号内部嵌套双引号，不取出变量值

1.3 ODS层日志表加载脚本

#!/bin/bash

APP=gmall

if [ $# -lt 1 ]
then
 exit
fi
do_date=$1
echo "==============日志日期为 $do_date ======="
sql="load data inpath '/origin_data/$APP/log/topic_log/$do_date' into table ${APP}.ods_log partition(dt='$do_date')";
hive -e "$sql"

2.28 ODS层业务数据首日导入脚本

#!/bin/bash

APP=gmall

# 如果是输入的日期按照取输入日期；如果没输入日期取当前时间的前一天
if [ -n "$2" ] ;then
   do_date=$2
else
